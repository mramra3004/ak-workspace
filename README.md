# NgEnt - Angular Project Structure

Each NgModule is a feature, and should apply the following structure ([core](./src/app/core)):

```
├── components
│   └── ...
├── constants
│   └── ...
├── containers
│   └── ...
├── feature.module.ts
├── data-services
│   └── ...
├── directives
│   └── ...
├── guards
│   └── ...
├── interceptors
│   └── ...
├── models
│   └── ...
├── pipes
│   └── ...
├── resolvers
│   └── ...
├── services
│   └── ...
└── view-models
    └── ...
```
